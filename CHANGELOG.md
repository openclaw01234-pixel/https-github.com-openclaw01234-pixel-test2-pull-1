# Changelog

## v1.0.0 - Initial Release

### Added
- Full single-page application with 7 views: Home, Player, Universal Search, Watch Party, Library, Vault, Settings
- Custom HTML5 video player with all standard controls
- AI-powered live subtitles using Web Speech API
- AI movie summary generator
- Smart scene skip
- Karaoke mode with synced lyrics
- Learn mode with word lookup dictionary
- VR split-view mode
- Watch Party with simulated bot chat and 8 floating reactions
- Universal search across 5 mock platforms (MX, Netflix, Prime, YouTube, Disney+)
- Mood-based recommendations (8 moods)
- Private Vault with PIN protection (default: 1234)
- Sleep timer (5m / 15m / 30m / 1h)
- Volume booster up to 250% via Web Audio API
- Picture-in-Picture support
- Screenshot capture
- A-B loop with custom set points
- Resume playback from last position
- Touch gestures: brightness/volume swipe, double-tap seek
- Full keyboard shortcuts: Space, Arrows, F, M, S, P, A, B
- Three themes: Dark, Light, OLED Black
- Responsive layout for mobile and desktop
- Local data persistence with localStorage

### Tech
- Vanilla HTML / CSS / JavaScript
- No build step required
- Browser APIs: Web Speech, Web Audio, MediaDevices, Picture-in-Picture, Fullscreen
